# 🤖 AI Trading Platform

An intelligent, real-time AI-powered trading assistant that combines **multi-source market data + technical indicators + LLM reasoning** to generate high-quality intraday trading signals.

---

## 🚀 Features

### 📡 Live Market Data (3 Sources)
- YFinance
- NSE Direct (jugaad-data)
- Yahoo Finance API

### 🧠 AI Trading Analyst
- Uses LLM (Groq models)
- Generates signals: **ENTER / HOLD / EXIT / WAIT**
- Provides:
  - Strike price
  - Entry, Stop Loss, Target
  - Risk-Reward ratio
  - Win/Loss probability

### 📊 Technical Indicators
- RSI (14)
- MACD
- SMA (20 / 50)
- Bollinger Bands

### 🎯 Smart Strike Selection
- ATM / ITM / OTM suggestions
- Based on RSI + trend

### 💰 Live P&L Tracking
- Index-based + option approximation
- Real-time updates

### 📰 Macro News Integration
- RBI policy updates
- Fed interest rate news
- Global market cues

### 📈 Interactive Charts
- Daily charts
- Intraday (5-min) charts
- RSI & MACD visualization
